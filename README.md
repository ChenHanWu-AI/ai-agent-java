**AI-Agent Java Framework**

一个基于 **Spring Boot** 的 AI 任务编排框架，通过 **自然语言指令 → 工具选择 → DAG任务调度 → 并发执行** 的方式，实现复杂任务自动化。

系统结合 **向量检索 + 大模型解析 + DAG调度引擎**，能够将用户的自然语言指令自动转化为结构化任务，并调用 Java 工具库执行。

------

# 核心架构

系统采用 **三层 AI 处理 + DAG 任务编排** 的架构：

### 1️⃣ 前置处理器（Instruction Preprocessor）

对用户输入的自然语言指令进行语义解析：

- 拆分为多个标准化子任务
- 补全隐含意图
- 生成结构化任务描述

# DAG任务编排

根据任务依赖关系构建 **有向无环图（DAG）**：

- 自动识别可并发执行任务
- 控制任务依赖顺序
- 传递前序任务执行结果

------

# 执行引擎

执行引擎基于：

- **Spring Boot**
- **Java 反射机制**
- **CompletableFuture 并发控制**

实现：

- 动态工具调用
- 并发任务执行
- 任务结果传递

------

# 工具库（Tool Library）

每个工具为独立 Java 方法，通过注解声明：

```
@Tool(
  name="sendWechatMessage",
  description="发送微信消息",
  dependsOn={"openWechat"}
)
```

特点：

- 支持运行时扩展
- 无侵入接入外部系统
- 自动被 AI 识别和调用

------

# 技术特点

**无侵入集成**

只需编写工具适配类，无需修改原系统代码。

**可扩展工具体系**

新增工具只需添加 Java 类和 `@Tool` 注解。

**语义驱动**

支持口语化、多意图复杂指令解析。

**任务编排能力**

内置 DAG 调度引擎，自动处理任务依赖与并发。

**Java 原生生态**

基于 **Spring Boot**，可复用企业现有 Java 代码。
